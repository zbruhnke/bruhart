# Bru-Hart SEO and SERP Audit

Date: April 26, 2026

Scope: Bru-Hart Industries, https://www.bruhart.com

## Executive Summary

The sitemap is not the limiting factor. The current public site is small and broad, while the search results for Bru-Hart's core commercial terms are won by specific product, application, and specification pages.

The main SEO constraint is limited URL depth for high-intent queries. Bru-Hart has a broad `/products` page and a broad `/industries` page, but searchers looking for "crash rated gates," "M50 barrier," "data center security gates," "utility substation security fencing," "cantilever gate track," or "gate access control" are usually served pages built around that exact intent.

Top opportunities:

1. Create dedicated product pages for crash rated gates, crash rated barriers, security fencing, gate access control, cantilever gate systems, and bollards/barriers.
2. Create industry pages for data centers, utility substations, airports, and government facilities where the product mix is explained by site risk and operations.
3. Build specification-support content around ASTM F2656, M-ratings, K-ratings, UL 325, and ASTM F2200, because ranking pages repeatedly explain standards and certification language.
4. Turn the custom fabrication offer into searchable component pages for enclosed aluminum gate track and CNC-milled gate trucks.
5. Add a separate local agricultural track for Branford and North Florida: agricultural fencing, field fence, horse fence, wire fence, farm gates, and fence materials.
6. Add proof assets over time: project photos, use cases, manufacturer documentation, specs, CAD/download links where appropriate, and FAQs that answer buyer objections.

Important positioning guardrail: Bru-Hart's gates should be described as crash rated, not crash tested. Do not claim physical crash testing unless a specific gate or barrier configuration has test documentation that supports that exact claim.

Local/agricultural positioning guardrail: the Branford agricultural opportunity is based on owner-provided local context and the company's stated land-use commitment, not Google Search Console data. Treat it as a real local business requirement, but avoid thin town-page SEO. Start with one strong Branford/North Florida service page plus specific agricultural product pages.
